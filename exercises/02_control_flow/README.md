# 🚦 Exercises: Module 2 - Control Flow

Practice problems for [`lessons/02_control_flow/`](../../lessons/02_control_flow/README.md):
conditionals and loops.

## 🧩 Tasks in `exercises.py`

- `fizzbuzz(n)` - classic conditionals-in-a-loop exercise: return
  "Fizz"/"Buzz"/"FizzBuzz"/the number itself for 1..n.
- `count_evens(numbers)` - count even numbers in a list using a loop and
  a conditional.
- `first_negative(numbers)` - find the first negative number in a list,
  looping with an early `return`.

## ▶️ How to work through it

1. Read [`lessons/02_control_flow/`](../../lessons/02_control_flow/README.md) first.
2. Open `exercises.py` and implement each function marked `# TODO`.
3. Run it:

   ```bash
   python exercises/02_control_flow/exercises.py
   ```

   It prints `All checks passed!` once everything is implemented
   correctly.
4. Compare with `solutions.py` if you get stuck or want a second opinion.
