# 🧪 Optional playground

Use this directory for temporary Python experiments that do not belong to a
module exercise. Create `main.py` locally if it does not already exist:

```python
print("Try one small idea here.")
```

Run it from the repository root:

```bash
python exercises/00_playground/main.py
```

`main.py` is ignored by Git, is not a trackable course unit, and is never used
as evaluator input. The learning tutor must preserve it as learner-owned work.
